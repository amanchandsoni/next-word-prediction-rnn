# Next Word Prediction using RNN

## Overview
This project predicts the next word in a sentence using a Recurrent Neural Network (RNN).

## Problem Statement
Next word prediction is a key task in natural language processing and is used in text generation systems.

## Approach
- Cleaned and tokenized text data
- Converted text into sequences
- Trained RNN model for sequence prediction

## Results
- Model predicts next word based on context
- Demonstrates understanding of language modeling

## Tech Stack
Python, TensorFlow/Keras, NLP
